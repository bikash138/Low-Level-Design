# Encapsulation

## What is Encapsulation?
It is somewhere same as Abstraction where it is used to hiding or protecting some data using  access modifiers.

## Types of Access Modifiers

### Public
Public access modifier means that the member is accessible from anywhere.

### Private
Private access modifier means that the member is accessible only within the class.

### Protected
Protected access modifier means that the member is accessible within the class and its derived classes.

## Difference between Encapsulation and Abstraction
- Encapsulation is used for data security using access modifiers.
- Abstraction is used to hide the implementation details and show only the functionality to the user.
